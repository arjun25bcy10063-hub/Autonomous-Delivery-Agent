Autonomous Delivery Agent

The agent's objective is to evaluate BFS, UCS, and A*'s performance (path cost, execution time, nodes expanded) quantitatively across a range of map sizes and complexity levels.  Demonstrate how an agent can change course in real time in response to obstacles that move.
 
Key Features:-

Obstacle Avoidance: By identifying walls and blocks, the agent automatically steers clear of obstacles.
Dynamic Grid: It is easy to alter the size of the city and the locations of obstacles.
